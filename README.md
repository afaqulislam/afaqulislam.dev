## Features

### Design & Visual
- **Dark Theme** with cyan (`#00d2ff`) and purple (`#7b2ff7`) gradient accents
- **Floating Particles** background with randomized colors and animations
- **Glass Morphism** navbar with backdrop blur on scroll
- **Gradient Text** headings using `background-clip`
- **Animated Profile Ring** with rotating orbital dots
- **Custom Scrollbar** matching the accent color theme
- **SVG Favicon** with "AUI" initials and gradient styling

### Animations & Interactions
- **Typed.js** — rotating roles (Frontend Developer, UI/UX Designer, Content Creator, Creative Thinker)
- **Scroll Reveal** — elements animate in as you scroll down
- **3D Card Tilt** — service and project cards respond to mouse movement
- **Magnetic Buttons** — buttons subtly follow the cursor on hover
- **Counter Animation** — stats count up when scrolled into view
- **Smooth Scrolling** — all anchor links scroll smoothly with offset
- **Back to Top** button appears after scrolling down
- **Hamburger Menu** — animated icon transforms to X on mobile

### Functionality
- **Active Nav Tracking** — highlights current section in navbar
- **Contact Form** — opens Gmail compose with pre-filled recipient, subject, and message
- **Dynamic Copyright Year** — auto-updates every year via JavaScript
- **Mobile-First Responsive** — works on all screen sizes

## Getting Started

1. **Clone the repository**

```bash
git clone https://github.com/afaqulislam/dev.afaqulislam.github.io.git
```

2. **Open in browser**

```bash
# Simply open index.html in your browser
open index.html
```

3. **Or run with Live Server** (VS Code)

```
Right-click index.html → Open with Live Server
```

No build tools, no dependencies, no compilation needed. It's pure vanilla web.
